Plan de Acción: Auditoría Técnica y Estratégica

Integración de los 10 puntos de auditoría + Planeación SEO + Solución de Duplicidad.

Área 1: SEO y Duplicidad

Riesgo: Contenido duplicado entre GitHub y el dominio pago.

Solución: Usar etiquetas Canónicas indicando que el dominio pago es el "Jefe". 
<!-- En el head de TODAS las páginas -->
<link rel="canonical" href="https://www.gramasyparquescolombia.com/productos/parques.html" />
<!-- Esto le dice a Google: 'Ignora la versión de GitHub, la oficial es esta' -->
Área 2: UX e Interactividad

Problema: El menú sticky generado por JS causa inestabilidad visual (CLS).

Solución: Definir el menú sticky por CSS puro para una carga instantánea. 
/* En style.css */
.category-nav { position: sticky; top: 0; z-index: 1000; background: white; }
Área 3: UI y Contraste

Problema: Botones naranja con poco contraste para lectura (Falla WCAG).

Solución: Ajustar el color a un naranja más profundo (#ea580c). 
.btn-quote { background: #ea580c; font-weight: 800; }
Área 4: Semántica HTML

Problema: Jerarquía de encabezados H1-H6 desordenada.

Solución: Usar un solo H1 con las keywords transaccionales del Excel. 
<h1>Grama Sintética <span>Económica</span> en Barranquilla</h1>
Área 5: Rendimiento CSS

Problema: Estilos incrustados en archivos HTML (Parques, Biosaludables).

Solución: Mover todos los estilos al archivo style.css global.
Área 6: Lógica JS

Problema: El modal de WhatsApp no limpia los datos tras el envío.

Solución: Resetear campos en main.js al hacer click en enviar. 
// En main.js
document.getElementById("wa-name").value = "";
document.getElementById("wa-desc").value = "";
Área 7: Velocidad (WPO)

Problema: Falta de Lazy Loading en imágenes del catálogo.

Solución: Añadir el atributo nativo de carga diferida. 
<img src="..." loading="lazy" alt="...">
Área 8: SEO de Imágenes

Problema: Atributos ALT genéricos como "Alejo" o "Banca".

Solución: Usar keywords específicas del catálogo: "Bancas de concreto", "Parques de madera".
Área 9: Seguridad de Datos

Problema: Formulario de cotización sin validación de campos obligatorios.

Solución: Añadir el atributo 'required' a los inputs del modal.
Área 10: Confianza (E-A-T)

Problema: No hay testimonios ni logos de clientes en la web.

Solución: Crear sección de logos de aliados (Alcaldías, Constructoras) antes del footer.
